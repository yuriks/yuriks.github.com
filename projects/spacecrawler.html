Space Crawler

Space Crawler is a game I'm working on in my spare time. It's a space combat/exploration game. The general idea is for it to be something akin to "Diablo in Space!", in that (at least some) missions will be procedurally generated and there will be many kinds of equipment to upgrade your spaceship.

Since development happens in my spare time, progress is going to be slow (but hopefully steady.) You can follow its development on the blog section of this website.

Downloads

Apart from the game itself, you're going to need the Visual C++ 2012 runtime, if you don't have it already, and an OpenGL 1.5 capable video card.

Controls for now are:

  • Left/Right arrows: Rotate ship
  • Up arrow: Thrust forward
  • Down arrow: Inertial brake
  • Z: Shoot

The game is written in C++, the source code is hosted on GitHub.
